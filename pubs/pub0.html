Propiedades CSS

A continuación se presentan algunas de las propiedades CSS más utilizadas para manejar imágenes de fondo y objetos:

Background Image

La propiedad background-image se utiliza para establecer una imagen de fondo para un elemento.

background-image: url('../assets/img/muestra.jpg');

Background Size

La propiedad background-size se utiliza para especificar el tamaño de la imagen de fondo.

background-size: cover;

Background Attachment

La propiedad background-attachment se utiliza para definir si la imagen de fondo se desplaza con el contenido o se fija en su lugar.

background-attachment: fixed;

Object Fit

La propiedad object-fit se utiliza para especificar cómo se debe ajustar una imagen dentro de su contenedor.

object-fit: cover;

Object Position

La propiedad object-position se utiliza para definir la posición de una imagen dentro de su contenedor.

object-position: top;

Object View Box

La propiedad object-view-box se utiliza para definir una caja de vista para una imagen.

object-view-box: inset(50% 50% 20% 20%);
